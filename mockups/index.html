Five takes on the same voice-first assistant. Same feature set in every one — voice conversation, a live observability feed, cross-project status, capture & triage, an action queue with confirm-gating, a model-tier slider, and a spend meter. Each shows a desktop view and an iPhone (iOS) view. These are exploration artifacts, not a commitment — review, then we pick (or mix).
